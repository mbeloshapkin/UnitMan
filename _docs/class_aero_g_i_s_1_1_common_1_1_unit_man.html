Detailed Description

Single C# class SDK for units of measurements conversion. The entire system of UOMs could be defined as single XML file. Redundant conversion coefficients effectively eliminated, UnitMan is able to caltulate conversion coefficients in many cases even these coefficients are not pre-defined.
